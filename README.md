# YouTube Planner

YouTube içerik üreticileri için geliştirilmiş not alma uygulaması.

## Özellikler

- Modern ve koyu tema arayüzü
- Zengin metin düzenleyici
- Sürükle-bırak ile not sıralama
- Yerel depolama ile veri saklama
- Duyarlı tasarım

## Teknolojiler

- React
- TypeScript
- TailwindCSS
- Tiptap Editor
- DnD Kit
